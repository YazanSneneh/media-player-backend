# Music Player Backend

## Prerequisite
* You need postgres to be installed on your server.
## Project Setup
1. clone repo.
2. cd to root of the project.

3. install required packages. 
```sh
npm install
```

4. Create `.env` in server root folder and put required keys
```env
PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_HOST=
DB_PORT=
```

### Compile and run

1. compile typescript:
```sh
npm run compile
```

2. start server

```sh
npm run dev
```
